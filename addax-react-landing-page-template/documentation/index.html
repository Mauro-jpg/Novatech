Addax - React Documentation

Introduction

Addax React is a creative and clean powerful responsive Landing Page pure React.js Template for presenting or introducing you or your business on the web! This template is offering multiple variations in landing page homepage design with unique User Experience (UX) and Look & Feel. It looks stunning on all types of screens and devices.

Setup Development Environment

In order to kickstart the development of the web app with Addax, the very first thing need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • Node JS works as a node package manager behind the scene. It is recommended to download and install the latest version from the Node JS official site https://nodejs.org/en/.

Preparation

You'll need to install Node.js (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project(Addax) root. This is powershell in windows and terminal on mac/linux.

Note:

This project set up with Express.js backend framework, so you have to decide what you do, run with express.js? or without express.js

Let's looking how setup with Express.js & without Express.js:
With Express.js

Just open terminal inside the project root & run: npm install after that move forward one directory such as client cd client and run npm install again that is for client, and again move back to one directory such ass cd .. now run npm run dev and this will return something like the following:

> addax@1.8.0 dev D:\tf-jsx\addax-react

> concurrently "npm run server" "npm run client"

> addax@1.8.0 server D:\tf-jsx\addax-react

> node index.js

> addax@1.8.0 client D:\tf-jsx\addax-react

> npm run start --prefix client

> Read on http://localhost:5000 (It is backend server url with port)

> addax-react@1.8.0 start D:\tf-jsx\addax-react\client

> react-scripts start

Compiled successfully!

You can now view addax in the browser.

Local: http://localhost:3000/

On Your Network: http://192.168.1.6:3000/

Now in the browser go to localhost:3000

Without Express.js

Just pick the client folder and inside that open the terminal and run: npm install and this will import all the dependencies for the Addax React. After that run npm start and this will return something like the following:

Starting the development server...

Compiled successfully!

Local: http://localhost:3000/

On Your Network: http://192.168.1.6:3000/

Now in the browser go to localhost:3000

Fonts

By default, the template loads Poppins font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.scss" file: /client/src/assets/css/style.scss


@import url('https://fonts.googleapis.com/css?family=#{$family}:100,100i,200,200i,300,300i,400,400i,500,500i,600,600i,700,700i,800,800i,900,900i');
